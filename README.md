# Veltra Online Store

Veltra is a simple React online store project.  
Product data is fetched from the Fake Store API.

## Features

- Browse products
- View product details
- View similar products
- Add products to cart
- Update item quantity
- Remove items from cart
- Clear cart
- View order summary
- Basic account and info pages

## Tech Used

- React
- React Router
- Axios
- JavaScript
- CSS
- Local Storage
- Fake Store API

## API

This project uses the Fake Store API:
https://fakestoreapi.com/products
